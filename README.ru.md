## О проекте

Conflux Arena — квест-платформа и система геймификации на блокчейне [Conflux eSpace](https://confluxnetwork.org). Пользователи подключают кошельки, выполняют ончейн- и социальные квесты, зарабатывают XP, соревнуются в лидербордах и получают NFT-награды.

Проекты запускают квесты для роста своих сообществ с верифицируемой ончейн-активностью — без ботов, без фейковой активности.

## Возможности

- **30+ типов квестов** — социальные (Twitter, Telegram, Discord), DeFi (swap, hold, stake, LP), NFT, викторины, реферальные
- **Ончейн-верификация** — DeFi и NFT квесты проверяются напрямую через блокчейн (RPC и ConfluxScan)
- **NFT-награды** — ERC-721 клейм с ECDSA-подписью, платный минт, настраиваемые периоды минта
- **Демо-трейдинг** — пара CFX/USDT0, плечо до 50x, виртуальный баланс, еженедельные соревнования
- **Прогнозы цен** — еженедельная игра на прогнозирование цены CFX с призовым пулом
- **Мультикошелёк** — MetaMask, Fluent, OKX, WalletConnect (500+ кошельков), EIP-6963
- **Гейтинг квестов** — контроль доступа по балансу токенов, владению NFT, уровню, стейкингу
- **Массовые выплаты** — мультитокен (CFX, USDT, USDC, PPI) батч-выплаты через смарт-контракт
- **Антифрод** — детекция дубликатов, кросс-кошельковый матчинг, анализ IP, скоринг бот-поведения
