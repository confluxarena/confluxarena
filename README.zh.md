## 关于

Conflux Arena 是一个基于 [Conflux eSpace](https://confluxnetwork.org) 区块链的任务与游戏化平台。用户连接钱包，完成链上和社交任务，赚取经验值，参与排行榜竞争，领取 NFT 奖励。

项目方通过发布任务来增长社区，所有参与均可链上验证——杜绝机器人和虚假活动。

## 功能

- **30+ 种任务类型** — 社交（Twitter、Telegram、Discord）、DeFi（swap、hold、stake、LP）、NFT、问答、推荐
- **链上验证** — DeFi 和 NFT 任务通过 RPC 和 ConfluxScan 直接在区块链上验证
- **NFT 奖励** — ERC-721 领取，ECDSA 签名验证，付费铸造，可配置铸造周期
- **模拟交易** — CFX/USDT0 交易对，最高 50 倍杠杆，虚拟余额，每周竞赛
- **价格预测** — 每周 CFX 价格预测游戏，设有奖金池
- **多钱包支持** — MetaMask、Fluent、OKX、WalletConnect（500+ 钱包）、EIP-6963 发现协议
- **任务门控** — 按代币余额、NFT 持有、等级、质押仓位控制访问
- **批量支付** — 多代币（CFX、USDT、USDC、PPI）通过智能合约批量发放奖励
- **反欺诈系统** — 重复检测、跨钱包证明匹配、IP 分析、机器人行为评分
